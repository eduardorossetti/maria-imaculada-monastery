# Mosteiro Maria Imaculada - Irmãs Clarissas

Blog do Mosteiro Maria Imaculada – site para irmãs clarissas.

## Stack

Next.js 16, React 19, TypeScript, Tailwind CSS v4, shadcn/ui, Zod v4

## Desenvolvimento

```bash
pnpm dev
```

## Build

```bash
pnpm build
pnpm start
```
